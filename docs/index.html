Key Financial Formula

Return on Invested Capital (ROIC):

\[\mathrm{ROIC} = \frac{\mathrm{NOPAT}}{\mathrm{Invested\ Capital}} = \frac{\mathrm{EBIT} \times (1 - t)}{\mathrm{Equity} + \mathrm{Debt} - \mathrm{Cash}}\]

Example: If NOPAT = 120M and Invested Capital = 800M, ROIC = 15%. \(\,\)

Quarter-over-Quarter Drivers

  1. Net Interest Income (NII) improvement
  2. Non-interest income — fees & commissions
  3. Cost control measures (efficiency ratio)

Efficiency Ratio formula: \(\mathrm{Efficiency} = \frac{Operating\ Expenses}{Net\ Revenue} \times 100\%\)

-- sample query used to compute NII
SELECT date_trunc('quarter', txn_date) AS qtr,
       SUM(interest_income) - SUM(interest_expense) AS nii
FROM ledger
GROUP BY 1
ORDER BY 1 DESC;
